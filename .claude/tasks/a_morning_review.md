# A股早盘行情复盘任务

> 执行时间：北京时间每日中午 12:00（早盘 9:30-11:30 已收盘）

## 第零步：更新代码仓库

```bash
git pull origin main || echo "git pull failed, continuing..."
```

## 第一步：读取前一天的预测记录

读取仓库 data/ 目录，找到最近一次的预测报告或信号文件，记录其中的主要预测观点（涨跌方向、重点板块、关键价位等）。如果找不到预测记录，说明未找到并继续。

## 第二步：获取今日A股早盘行情

使用仓库中可用的工具和技能获取今日早盘行情数据，包括：
- 上证指数、深证成指、创业板指的涨跌幅
- 今日热点板块及涨幅前列个股
- 成交量与资金流向
- 北向资金动向（如可获取）

## 第三步：对比分析

将实际行情与前一天预测逐条对比：
- 预测准确的部分（✓）
- 预测偏差的部分及原因分析（✗）
- 整体预测命中率评估

## 第四步：生成复盘报告

将报告保存到 data/daily_review/ 目录，文件名格式：YYYY-MM-DD_morning_review.md（使用今天实际日期）。

报告结构：
1. 今日早盘概览
2. 前日预测回顾（如有）
3. 预测 vs 实际对比表
4. 偏差原因分析
5. 对下午盘的参考启示

完成后输出报告内容摘要。

## 最后步骤：提交并推送至GitHub

```bash
git add data/
git commit -m "auto: A股早盘复盘 $(date +%Y-%m-%d)" || echo "nothing to commit"
for i in 1 2 3; do git push origin main && break || { echo "retry $i"; sleep 10; }; done
git log --oneline -1
git status
```

- 成功：「✅ 分析结果已成功推送至GitHub」
- 失败：「❌ 推送失败（已重试3次），错误：[详细错误信息]」
