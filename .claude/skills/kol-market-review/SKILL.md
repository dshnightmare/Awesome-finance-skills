# KOL Market Review Skill

## Overview

对当日A股行情，以7位知乎财经KOL的真实投资框架逐一进行视角分析，最终汇总共识与分歧，形成结构化复盘报告。

**7位KOL：**
1. **么瑞与小A** — 技术波段 + 仓位管理
2. **MR Dang** — 宏观数据解读 + 事件分析
3. **寒武纪的鳄鱼** — 周期×价值×极致长持
4. **派大星皮皮** — 趋势配置 + 去弱留强
5. **Deep Van（杨磊）** — 全球产业链 + 地缘宏观 + 多空对冲
6. **白白胖胖0（龙方）** — 估值筛选 + 技术右侧入场
7. **龙开** — 宏观叙事 → 期货量化执行

---

## Capabilities

### 1. 全量KOL复盘（默认）

对当日行情从7个视角全量分析，输出综合复盘报告。

**触发词**：复盘今日行情 / 今日A股复盘 / 7个视角分析

### 2. 单KOL视角分析

只用指定KOL的框架分析当日行情。

**触发词**：用{KOL名}的方法分析今天 / 从派大星视角看今天

### 3. 次日预测 + KOL框架辅助

结合已有的A股预测方法论（见 `feedback_astock_prediction_methodology.md`），用各KOL框架提供补充视角。

**触发词**：预测明天 / 明日研判

---

## Agentic Workflow

### Step 1：获取当日市场数据

运行 `scripts/fetch_market_data.py` 获取基础行情数据，或直接读取用户提供的行情信息：

```bash
/opt/miniconda3/bin/python3 scripts/fetch_market_data.py             # 今日
/opt/miniconda3/bin/python3 scripts/fetch_market_data.py --date YYYY-MM-DD  # 指定日期
/opt/miniconda3/bin/python3 scripts/fetch_market_data.py --json      # JSON格式供程序读取
```

输出包含：各大指数涨跌幅、成交额、板块TOP5涨跌、北向资金、个股涨跌家数等。

### Step 2：读取KOL框架

读取 `references/kol_frameworks.md`，获取每个KOL的分析清单和核心问题。

### Step 3：逐KOL填写分析

对每位KOL，按其框架回答所有核心问题，生成该视角下的行情判断。

**重要原则**：
- 每个KOL只用自己关注的变量，不混用
- 结论要量化（如仓位建议用成数，概率用%）
- 对KOL不关注的维度，标注"该KOL不关注此项"

### Step 4：汇总共识与分歧

提炼7个视角中的：
- **共识**：至少5/7视角一致的判断
- **分歧**：视角间最大的判断差异
- **关键分歧点**：哪条信息决定了视角分歧

### Step 5：输出报告

按以下结构输出Markdown报告。
