# KOL投资方法论框架库

本文档将7位知乎财经KOL的真实投资方法论结构化为可执行的复盘清单。  
每个KOL包含：核心框架、复盘问题清单、信号触发条件、输出格式。

**数据来源**：2025年11月–2026年5月共6个月知乎动态实录分析

## KOL 1：么瑞与小A

**定位**：A股波段操盘手，每日公开仓位，技术系统化交易者  
**年化目标**：15%，不追暴利，追求低波动复利

### 核心框架

```
观察量能变化 → 识别K线形态 → 判断蝗虫过境风险 → 决定加减仓方向
```

**仓位范围**：0成（空仓）~ 9成，半年多数时间在2–6成  
**核心理念**：确定性是唯一加仓理由；不确定时宁可踏空

### 信号体系

| 信号类型 | 看涨信号 | 看跌/警惕信号 |
|---------|---------|------------|
| 量能 | 放量上涨 / 缩量调整后放量突破 | 放量滞涨 / 量价背离 |
| K线形态 | 收盘价=最高价（尾盘拉升） / 快速下跌后十字星 | 长上影线 / 收盘价=最低价 |
| 缺口 | 缺口补完后反弹 | 未补缺口持续高位运行 |
| 板块 | 强势板块扩散至次强板块 | 只有1-2板块涨，余皆绿 |

### 蝗虫过境识别（核心风险信号）

**触发条件（任意一条）**：
- 某板块单日涨幅 > 5%，且前5日累计涨幅 > 20%
- 成交额短期暴增后迅速萎缩
- 多个无关联板块在同一天爆发（资金轮动收割特征）
- 量化主力特征：早盘拉升 + 午后砸盘

**已确认的蝗虫模式板块（历史案例）**：  
商业航天（2025年底）、AI应用（2026年初）、贵金属（2026年Q1）、电力（2026年3月）、CPO（2026年Q2）

### 仓位决策规则

```
空仓 → 加仓条件：
  - 连续快速下跌3天以上（暴力反弹概率提升）
  - 十字星出现在低位（变盘信号）
  - 确认非蝗虫过境行情

持仓 → 减仓条件：
  - 指数连续上涨后出现顶部结构（两根上影线）
  - 主力板块出现蝗虫过境信号
  - 今日高开后冲高回落（全天涨幅被吃掉）

维持现仓：
  - 震荡横盘，方向不明
  - 量能未放大，行情真空期
```

### 复盘问题清单

1. 今日三市成交额是多少？较昨日放量/缩量多少亿？
2. 中证1000今日K线形态是什么？收盘价在日内区间哪里？
3. 今日领涨板块是否有蝗虫过境特征？
4. 是否出现顶部结构（两根或以上上影线）？
5. 目前仓位（几成）对应的策略是否需要调整？
6. 今日有无异常拉升后砸盘的板块（量化嫌疑）？

### 输出格式

```
【么瑞与小A视角】
量能：X.XX万亿，较昨日 +/- XXXX亿（放量/缩量）
形态信号：（描述）
蝗虫风险：高/中/低（针对板块：XXX）
仓位建议：X成（原因：）
操作方向：加仓/减仓/维持/清仓
```

## KOL 2：MR Dang

**定位**：宏观分析师，不直接给操作建议，专注把经济事件转化为投资逻辑  
**核心方法**：事件串联 → 利益归属分析 → 板块传导链推演

### 核心框架

```
识别当日核心事件 → 串联上下文 → 判断利益归属 → 推演板块传导 → 识别市场定价偏差
```

**信条**：
- "新闻要连起来看，不能孤立解读单条消息"
- "年报是用来排除错误选项的，不是用来确认正确答案的"
- 市场定价偏差才是真正的机会

### 事件解读框架

**第一步：事件分类**

| 事件类型 | 分析重点 | 常见偏差 |
|---------|---------|---------|
| 经济数据（CPI/PPI/GDP/贸易） | 结构比总量重要；环比改善vs同比 | 只看总量，忽略结构性机会 |
| 美联储/货币政策 | 措辞变化（如"进一步"暗示降息） / 反对票数分布 | 只看最终结论，忽略内部分歧 |
| 地缘政治 | 利益归属分析（谁受益，谁受损） | 统一套用"不确定=震荡" |
| 企业财报 | 扣除非经常性损益的真实盈利；管理层指引 | 只看净利润，忽略质量 |
| 产业政策 | 受益链条（直接+间接）；政策时间窗口 | 只买直接受益，错过传导链 |

**第二步：利益归属分析（地缘事件专用）**

| 事件性质 | 受益方 | A股信号 |
|---------|-------|--------|
| 双边共同让步（关税互降、贸易协议） | 双方 | 整体偏多 |
| 外方来华拿订单/市场 | 外方企业 | 国产替代承压，科技偏空 |
| 地缘冲突缓和 | 全球 | 风险偏好改善，周期偏多 |
| 制裁/出口管制升级 | 无 | 偏空 |
| 峰会进行中，结果未知 | 不确定 | 震荡扩大，方向未定 |

**第三步：板块传导链推演**

以经济数据为例：
```
PPI超预期上涨 → 生产成本上升压力 → 中游化工承压
              → 上游资源受益（有色、能化）
              → 滞后3-5个月传导至CPI → 通胀预期 → 黄金受益
```

### 分析质量指标

- **二阶思考**：看到消息后，先问"市场会怎么解读"，再问"这个解读正确吗"
- **定价窗口**：事件是否已被提前1-3天定价？如是，当日信息价值打折
- **矛盾识别**：当权威方（政府/央行/官员）说法相互矛盾时，追踪利益而非说辞

### 复盘问题清单

1. 今日有哪些核心宏观事件/数据发布？
2. 市场对这些事件的反应是否存在定价偏差？
3. 地缘事件（如有）的利益归属是什么？A股是受益还是受损？
4. 今日发布的经济数据结构如何？哪些细分行业环比改善？
5. 是否有重要公司财报？关键数字是否被市场正确解读？
6. 今日市场反应与宏观逻辑是否一致？若不一致，是机会还是陷阱？

### 输出格式

```
【MR Dang视角】
今日核心事件：
  1. （事件描述）→ 利益归属：（谁受益）→ 板块信号：（涨/跌/中性）
  2. ...

市场定价偏差：（描述市场解读和真实逻辑的差距）
受影响板块：
  受益：
  受损：
  延迟传导（1-3个月后）：
关注信号：（下一个数据发布点/事件节点）
```

## KOL 3：寒武纪的鳄鱼

**定位**：极致长持型深度价值投资者，5年验证账户：1000万→5500万  
**信条**："一动不动像极了我们的投资" / "买入之时，胜负已定"

### 核心框架（三支柱）

```
周期（最重要）× 价值 × 复利
```

1. **周期**：所有资产价格遵从周期，在底部建仓，顶部离场
2. **价值**：用5毛钱买1块钱的东西（至少50%安全边际）
3. **复利**：平均持仓570天，时间是朋友，频繁交易是敌人

### 选股硬约束

- **只买央企**（97家央企及其300+上市子公司）
- **只买资源/大宗商品类**（金属矿、化工、石化、能源）
- **永不碰民企**（2018年浮工互动亏500万，铁律从此立）
- **永不追高**（买在月线/周线历史低位区域）

### 买入信号

```
触发条件（同时满足）：
1. 月线图：股价处于历史低位区（视觉判断，K线图"一眼低"）
2. 周线图：同样在低位，无大量未出清套牢盘
3. 行业景气：处于周期底部（整行业亏损或低利润，产能退出中）
4. 公司质量：央企资质，业务是基础物资，无法被淘汰
5. 估值：PB < 1 或股价 < 净资产，股息率 > 银行利率2倍
```

**代表性估值案例**：
- 中石化5.28元：净资产6.89元，30%折价，股息率5%+
- 金钼5.5元（2021年）：月线历史低位，钼价格处于底部

### 卖出信号

```
卖出条件（满足任意一条）：
1. 股价涨到"连自己都觉得不正常"的程度（主观感受，市场疯狂时）
2. 行业景气恢复到顶部区域（全行业高利润，产能快速扩张）
3. 时间窗口：大宗商品行业周期顶部通常在景气3-5年后
```

**案例**：金钼5.5→27元，持仓4.5年；"再过几年，等金属周期走完再买回来"

### 持仓管理

- 跟踪约50-60只央企资源股，但实际持仓高度集中（1-3只重仓）
- 持仓期间：看书、睡觉、等待，极少看盘
- 大跌时买、无聊时等、暴涨时卖
- 止损观：无止损（"需要止损说明买入就错了"）

### 复盘问题清单

1. 今日央企资源板块（有色/化工/石化/能源）整体表现如何？
2. 有没有哪个板块出现在月线/周线历史低位且今日异动？
3. 今日大宗商品价格变化（钼/铜/糖/石油/化工品）？
4. 当前该板块是周期底部还是顶部区域？（用行业利润率判断）
5. 今日热门板块（科技/AI/概念股）的表现如何？（反向信号：越热越要等）
6. 是否有新的低位央企资源股出现买入信号？

### 输出格式

```
【寒武纪的鳄鱼视角】
今日央企资源板块：
  有色：+/-X%（代表股：XXX）
  化工：+/-X%（代表股：XXX）
  石化/能源：+/-X%

周期位置判断：底部/中段/顶部（依据：行业利润率、产能状态）
今日买入机会：有/无
  若有：（股票、当前价、估值、信号）
持有逻辑更新：（是否有需要重新评估的信息）
市场噪音屏蔽：今日热门概念XXX与我无关，无需关注
```

## KOL 4：派大星皮皮

**定位**：全市场趋势配置者，每日三评（早/午/收），公开持仓和调仓  
**账户特征**：长期接近满仓，多次创历史新高

### 核心框架（三问）

每次操作前必须回答：

```
Q1. 现在是牛市/震荡市/熊市？
Q2. 现在是增量/存量/减量市场？
Q3. 短期/中期/长期是看多还是看空？

→ 答案决定仓位级别：
  牛市+增量+看多 → 满仓进攻型配置
  震荡+存量 → 高仓位+结构性防御
  熊市/减量 → 降仓位，防御优先
```

### 指数优先级体系（核心判断）

```
底仓排序：中证500 > 科创50 > 创业板 ≈ 沪深300
原因：
  - 中证500：最大机会/风险平衡点，国家队非主要目标
  - 科创50：AI+半导体主升浪核心载体，历史新高可能性高
  - 创业板：成长股聚集，但高弹性需要管理节奏
  - 沪深300/上证50：国家队砸盘主要对象，绕着走
```

### 进攻板块框架

**主线筛选标准**：
1. "十五五规划"重点方向中有真技术壁垒的
2. 全年高景气但已充分调整的（性价比高）
3. 业绩空窗期（5-7月）有题材催化的

**2026年活跃板块排序**（更新至2026年5月）：
```
商业航天 ≈ 钨（小金属） > 国产半导体设备 ≈ 户储 ≈ 化工ETF
> 卫星ETF ≈ 钢炼术士 ≈ 港股创新药
防御仓：红利低波ETF（行情不好时停车场）
```

### 去弱留强操作规则

```
触发去弱条件（任意一条）：
  - 持仓标的连续3日表现弱于同类
  - 持仓方向出现成交量萎缩+价格下跌（资金离场信号）
  - 单标的仓位超过上限（通常20-25%）

去弱后留强：
  - 资金转移到同期表现最强且有持续性的方向
  - 不追最热板块，已在里面的继续持有
  - 关注"没爆量的方向"（还有上涨空间）
```

### 成交量核心规则

```
3万亿以上 → 高仓位运行，进攻姿势
2-3万亿   → 中性仓位，结构性机会
<2万亿    → 防御，减少交易频率

放量上涨 → 有效突破，可追
缩量上涨 → 锁仓型涨，持有不追
放量下跌 → 出货信号，需判断
缩量下跌 → 洗盘可能，观察
```

### 融资盘信号

融资余额大幅增加 + 板块加速上涨 → 警报：情绪过热，融资客心态不稳，大概率拿不住，短期顶部风险

### 复盘问题清单

1. 三问答案是什么？（今日定性）
2. 今日成交额是多少？对应仓位级别？
3. 指数排序中哪个表现最强/最弱？
4. 活跃方向清单中哪些方向今日资金流入明显？
5. 有哪些持仓方向需要"去弱"？
6. 融资盘有没有快速增加的板块（过热信号）？
7. 国家队今日是否砸盘300/50ETF？

### 输出格式

```
【派大星皮皮视角】
三问答案：
  Q1 市场性质：牛市/震荡/熊市
  Q2 资金性质：增量/存量/减量（成交额：X.XX万亿）
  Q3 方向：短期看多/空 + 中期看多/空

指数表现排序：（按今日涨幅排列）
进攻方向评估：
  优选：
  次选：
  回避：
去弱留强信号：（哪个方向需要减，资金去哪里）
明日关注：
```

## KOL 5：Deep Van（杨磊）

**定位**：全球多资产配置 + 量化私募基金经理，"叫兽指数"管理者  
**核心优势**：产业链穿透 + 地缘-资产关联 + 多空对冲组合

### 核心框架（四问筛选）

只有同时满足以下四条才值得重仓：

```
1. 能否并入美国AI产业链？（全球需求验证）
2. 能否出海？（不依赖单一内需市场）
3. 内卷不？（行业竞争格局，有无定价权）
4. 有没有巨大的政策风险？（补贴依赖=政策风险）
```

**逆向推导法**：从终端需求出发，逆向找最稀缺的供给节点
```
AI大模型训练 → 算力需求 → HBM内存需求
           → HBM比DRAM技术壁垒高2倍以上
           → SK Hynix市占第一 → 用2x杠杆ETF放大
           → 混合键合设备（BESI）→ 前道设备（东京电子、ENTG）
```

### 地缘-资产关联框架

| 地缘状态 | 做多方向 | 做空方向 |
|---------|---------|---------|
| 中东冲突激化 | 战争金属（钨：ALM/中钨高新）+ 能源（XLE）+ 黄金 | 科技/风险资产 |
| 和平预期升温 | AI/科技（QQQ/半导体）+ 港股科技 | 能源/地缘受益板块 |
| 美元走强 | 黄金对冲（适量）| 新兴市场/港股 |
| 美联储降息预期 | 成长股/AI | 金融价值股 |

**多空配置原则**：
- 多头：出海+AI+全球需求确定
- 空头：内卷+补贴依赖+股东回报文化差的市场（恒生科技、茅台等）
- 净敞口：根据市场状态主动管理（牛市偏多，不确定期降低净敞口）

### 股东回报质量判断

**优质市场标准**（Deep Van核心价值观）：
- 高ROE + 持续分红 + 积极回购
- 日韩市场逐渐改善（受企业治理改革推动）
- A股和港股：股东回报文化较弱，需要选优质个例

**做空逻辑触发条件**：
- 恒生科技：估值无法支撑 + 无股东回报文化 → 持续2x反向
- 白酒（茅台/五粮液）：消费降级 + 出口受限 + 内卷 → 做空

### 期权使用原则

```
期权功能：尾部风险对冲（非投机）
买入Put：当核心持仓大幅盈利后，用小比例期权锁定利润
买入Call：重大催化前低成本博取上涨空间
核心原则：期权费总量不超过组合的2-3%
```

### 复盘问题清单

1. 全球AI产业链今日信号：美股半导体/HBM/算力方向涨跌？
2. A股对应标的（寒武纪、海光等）是否跟随还是背离？
3. 地缘局势今日有何变化？对应资产类别信号是什么？
4. 是否有新的"并入AI产业链/出海"机会出现？
5. 做空标的（恒生科技/白酒等）今日状态是否需要调整？
6. 股东回报信号：今日有无大额回购或分红公告？
7. 美元/美债实际利率变化？黄金信号？

### 输出格式

```
【Deep Van视角】
全球产业链信号：
  AI/半导体：美股 +/-X% → A股对应 +/-X%（传导/背离）
  HBM/存储：
  地缘相关（钨/能源）：

地缘状态更新：（中东/中美/台海）→ 资产信号
多头方向评估：（哪些符合四问？）
空头方向评估：（是否需要调整对冲比例）
组合净敞口建议：（偏多/中性/偏空）
```

## KOL 6：白白胖胖0（龙方）

**定位**：职业散户，A股7年+，基本面筛选+技术右侧入场  
**核心能力**：估值精算 + 严格止损纪律 + 独立判断管理层预期

### 核心框架（三步法）

```
第一步：基本价值线筛选（排雷）
  扣非EPS × 合理PE区间 = 合理价值下限
  只考虑扣非净利润连续向上的龙头

第二步：左侧网格降低成本
  低估值+景气向上 → 分批网格建仓
  逻辑：不知何时起涨，网格降本同时不错过行情

第三步：右侧技术面触发
  突破回调波段1/2 → 右侧试错入场
  "33公式"：21日内强势5天 → 主力意愿强 → 可以介入
```

### 技术分析工具集

| 工具 | 用途 | 注意事项 |

| 波浪理论 | 识别波段高低点 | 需量价节点配合，不单独使用 |

### 止损纪律（最核心）

```
价格止损：跌破关键支撑位收盘前机械执行，无例外
时间止损：持仓8天内盈利不超过10% → 止损离场（无效持仓）
心理原则："止损比止盈重要10倍"
绝不补仓：跌了不加，越跌越补是散户最大的错误
```

## KOL 7：龙开

**定位**：期货自营出身（豆油期现套利），转为量化个人交易者，账户千万级  
**核心工具**：宏观叙事 + 中金4因子黄金模型（改进版）+ 机械止损

### 核心框架

```
宏观叙事建构（自上而下）
  ↓
量化模型验证（中金4因子）
  ↓
期货/期权重仓执行
  ↓
机械止损（每手3万，触发即平）
```

### 宏观叙事框架

**当前（2026年5月）宏观叙事**：
```
地缘冲突（中东）→ 原油高位 → 全球大通胀
美联储 → 受政治压力，加息受限
中国央行 → 内需弱，宽松偏向
结论：宽松+通胀 = 资产价格上涨（金/银/股）
```

**叙事更新触发条件**：
- 美联储超预期加息 → 叙事转向，黄金/股票风险上升
- 地缘冲突结束（霍尔木兹重新开放）→ 原油下跌，通胀逻辑弱化
- 中国内需大幅好转 → 央行收紧，股指多头谨慎

### 黄金模型（中金4因子改进版）

```
黄金价格 = f(美债实际利率, 美元指数, 央行购金规模, 美国债务规模/GDP)

信号解读：
- 美债实际利率下降 → 持有黄金机会成本降低 → 多黄金
- 美元走弱 → 黄金（美元计价）上涨 → 多黄金
- 央行购金加速（如央行月购量翻倍）→ 需求侧驱动 → 多黄金
- 美国债务占GDP持续升高 → 美元信用风险 → 多黄金
```

### 主要交易品种与逻辑

| 品种 | 核心逻辑 | 操作方式 | 止损 |
|------|---------|---------|------|
| 沪金 | 大通胀+美债贬值+央行购金 | 最重仓，分批建仓 | 每手3万 |
| 上证50股指 | 慢牛+年报业绩+降息预期 | 中长期持有 | 每手3万 |
| 白银 | 太阳能工业需求+黄金溢出效应 | 短线辅助 | 每手3万 |
| 原油 | 霍尔木兹封锁+战争溢价 | 事件驱动短线 | 每手3万 |

### 止盈原则

```
"买预期，卖现实"：
  - 事件预期阶段：持有多单
  - 事件落地当日：立即止盈（无论盈亏）
  
案例：战争消息导致油价大涨 → 当日或次日平多
      美联储宣布降息 → 宣布当天平黄金多单
```

### 风险特征（真实数据）

研究期内（6个月）单次极端案例：
- 最大亏损：沪金单笔 -183万 + 股指 -111万（超买后回调）
- 最大盈利：沪金单笔 +170万+（周期操作）
- 特征：高波动、高弹性、接受大幅回撤以换取大级别行情

### 复盘问题清单

1. 宏观叙事今日有什么更新？（通胀/利率/地缘状态）
2. 中金4因子今日变化：美债实际利率/美元/央行购金/美债规模？
3. 黄金模型信号：多/空/中性？

5. 机械止损检查：持仓品种是否有触发每手3万止损的？
6. "买预期卖现实"检查：当前持有的多单是基于什么预期？预期是否已经落地？
7. 今日地缘新闻的实质性内容是什么？（区分噪音和真实变化）
